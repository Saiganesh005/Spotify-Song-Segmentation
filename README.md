# 🎵 Spotify Songs Genre Segmentation using Machine Learning & Deep Learning

## 📌 Project Overview

This project clusters Spotify songs based on their audio features using Machine Learning (K-Means Clustering) and Deep Learning (Autoencoder). It also recommends similar songs using cluster information.

---

## Features

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Scaling
- K-Means Clustering
- Deep Learning Autoencoder
- PCA Visualization
- Recommendation System
- Streamlit Deployment
